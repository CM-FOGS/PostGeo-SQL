The project is a set of vector geospatial functions designed to solve simple and complex geospatial problems in the PostgreSQL and PostGIS DBMS software environment.
PostgreSQL programming language. 
A brief description of how geo-functions work is generally published on the Q&A website at the following link - https://gis.stackexchange.com.Geo-functions.
Function names labeled - experimental, work only in the construction - WITH Queries.
